预约口罩途径

  • 漱玉平民防疫物品预约 【微信小程序】（每次3个，隔3天再约下一次） ​
  • 爱城市网【APP】（每次5个，隔3天再约下一次）

网络请求分析

漱玉

请求信息

需要 地区、街道、社区、身份证号、姓名、手机号、坐标、药店号、openid
openid是记录一个微信号在一个小程序的唯一标识

错误码

  • 40001 今日预约量上限
  • 40003 账号（openid）三日内有预约记录
  • 40006 身份证号三日内有预约记录

爱城市网

请求信息

需要 手机号、身份证号、姓名、药店信息、token
token是记录当前的登录信息

openid和token获取

安卓

安卓7.0以上因为系统的安全策略不能抓https的数据，解决方案比较麻烦安卓7.0以下可以用Packet Capture这个APP，或者用下面的Charles

ios

Charles软件需要在电脑上安装，代理网络请求教程 https://www.jianshu.com/p/5539599c7a25​

过滤

漱玉的服务器是 shunneng.com
openid在请求的内容里
爱城市的服务器是icity24.cn
token（Access_token）在post请求的​header里​

伪造请求

将获取的信息写入配置文件
用python的request包伪造请求，代码和药店信息在这里
https://pan.baidu.com/s/1ihEbHugE-GNjwGlwBiWLEQ 提取码：unpw
