【论文】基于概念图的LPAG算法for ALS

发表于 2019-05-28 | 更新于: 2019-07-23 | 阅读次数:
字数统计:

A Concept Map-Based Learning Paths Automatic Generation Algorithm for Adaptive Learning Systems
山东师范大学 山东女子学院 IEEE Access 李言从 王晓 邵增珍 2018.12

摘要

概念图是自适应学习系统中重要的知识可视化工具。通过对概念图的分析，找出学生的薄弱概念，为学生和教师提供学习指导和教学建议。然而，在目前的研究中，当概念数量较大且概念之间的关联复杂时，很难从概念图中分析出有效的信息。它很少能反映不同学生群体的学习成绩，不能反映自适应学习系统的特点。针对自适应学习系统，提出了一种学习路径自动生成算法——学习路径生成(LPG)算法。LPG算法充分考虑了不同学生群体的学习性能。通过聚类算法和关联规则挖掘，生成具有不同学习特征的概念图，利用拓扑排序算法生成多个简化的学习路径。实验表明，LPG算法对学生群体有较好的识别能力，可以根据学生的学习成绩生成不同的学习路径。

介绍

自适应学习系统在电子学习和智能教育领域得到了广泛的研究。它是一个可以根据学生对建模的结果进行自适应调整的学习系统。概念图作为自适应学习系统中有效的知识可视化工具，已成为当前研究的热点。概念图能够以结构化的形式表达概念，既能促进学生的有意义学习，又能根据学生的学习表现对概念结构进行自适应调整。

研究人员对概念图进行了大量的研究，并提出了许多由概念图派生的方法。

  1. Lee等人使用关联规则挖掘算法分析概念之间的关联规则，并自动生成概念图。基于这些自动生成的概念图，为学生建立了一个补救教学路径(RIP)。然而，他们所建立的教学路径是针对所有被他们分析过的学生。
  2. Atapattu等人[7]提出了一种利用自然语言处理技术从幻灯片中生成概念图的自然语言处理(NLP)算法，作为专家手工生成概念图的主动替代方法。然而，生成的概念图是静态的，不能反映学生的学习成绩。
  3. Huang等人使用分类方法对学生进行分组，并基于关联规则法生成概念图。生成的概念图反映了学生的整体表现，不具备区分学生的能力。
  4. Acharya等人提出了一种基于概念图的诊断学习方法，利用直接哈希和剪枝算法构造概念图。删除概念图中的冗余，生成一个学习序列。基于这个学习序列利用Android模拟器开发了一个原型学习系统。这种学习顺序并不能反映不同学生群体对不同概念的掌握情况。

总的来说，研究者们基于概念图进行了大量的研究，并取得了许多成果。然而，这些研究也有一些局限性。这些研究没有根据学生的学习行为数据来区分和分析学生。所有学生都在统一分析的指导下学习，不能反映适应性学习系统的特点。进一步分析概念图主要依靠劳动力。当概念之间关联规则的数量增加时，很难从概念图中提取有效的信息。针对自适应学习系统，提出了一种基于概念图的学习路径自动生成算法LPG(学习路径生成)。

LPG算法针对的就是上述局限性。它是基于学生的掌握水平的计算机科学的概念,利用聚类和关联规则挖掘算法来生成概念图与不同的学生的学习特点,并利用拓扑排序算法自动生成学习路径,这是根据自适应学习系统的特点。

本研究的主要贡献如下:

  • (一)根据学生的学习成绩数据生成若干概念图。本研究可以克服现有研究对学生学习行为不区分的局限性。
  • (二)将学生按不同的学习特点分组，每组学生分别进行分析。分组后的学生可以通过不同的方式进行指导。学生分组后学习概念的时间长短不同，反映了LPG算法的适应性。
  • (三)将拓扑排序算法应用于概念图分析，自动生成多个不同的学习路径。学习路径的生成不需要劳动力，节省了劳动力成本。教师可以根据学习路径安排教学顺序和教学时长，每组学生都有不同的教学方法指导。

相关工作

康奈尔大学的Novak和Gowin[10]首次提出了概念图，用节点表示概念，用有向边表示概念之间的关联规则。概念图在自适应学习系统[11]中得到了广泛的应用。自适应学习系统是一种考虑学生学习成绩，强调个性化学习的系统。在自适应学习系统[12]中，概念图作为一种直观的知识可视化工具，可以根据学生的学习表现进行自适应调整。通过对概念图[13]、[14]的分析，可以挖掘出有价值的信息。

自概念图提出以来，研究者们对概念图的生成和应用进行了大量的研究。Chen等人通过学术文章为电子学习生成了概念图。他们使用电子学习期刊文章和会议论文作为数据源，并使用文本挖掘技术自动生成概念图。Qasim等人提出了一种基于集群的方法，用于从文本文档生成概念图。他们使用了多种技术来处理文本文档，如自然语言处理、聚类、信息检索和结构相似性度量。生成的概念图可以辅助教学。Lee等人[17]提出了一种利用单词的爆发从单个文档自动生成概念图的方法，该方法得到了验证。Wang等人提出了一种从教科书中生成概念图的联合优化模型。该模型使用教科书中隐含的概念关键字之间的关系生成概念图。Lai等人[19]提出了一种基于信息检索技术的系统，该系统可以从书的各个部分提取关键词，生成关于这本书的关键词概念图。Santos[20]对基于自然语言处理和机器学习技术的概念进行了抽象分类，并分析了概念之间的关系，自动生成概念图。

这些研究使用文本分析、自然语言处理、机器学习和其他技术来分析非结构化文本并自动生成概念图。生成的概念图可以为教师提供有用的参考，为教师设计教材，帮助学生理解概念的结构。这些研究从文本中提取关键项目并生成概念图。但是，生成的概念图并没有分析学生的学习行为，也没有反映学生对概念的掌握程度，难以因材施教。

也有很多论文考虑学生的学习行为，生成和分析概念图。Lee等人提出了一种利用Apriori算法生成概念图并利用概念图诊断教学障碍的系统。该系统考虑了学生的学习成绩，对学生进行了统一的指导。Bai等人提出了一种基于模糊规则[22]和学生答题记录的概念图自动构建方法。利用模糊规则和模糊推理技术，自动构造概念图，并对概念之间的关联度进行评价。生成的概念图表明，应该先学习一个概念，然后再学习另一个概念。Chen等人提出了一种基于数据挖掘技术的自适应学习系统概念图的自动生成方法。该方法可以根据学生的答题记录动态生成概念图，具有一定的参考意义。Huang等人使用关联规则法对所有学生的考试记录进行分析，自动生成概念图。生成的概念图反映了所有学生的学习成绩。Schroeder等人研究了使用概念图学习的效果。他们将概念图应用于所有学生，并验证了概念图在学习上具有持续的优势。Romero等人提出了一种基于概念图的学习策略。该策略由学生创建概念图，并与教师创建的概念图进行比较，发现学生在概念掌握上的弱点。该策略注重学生的学习体验，具有主观性。这些研究取得了很大的成功，符合自适应学习系统的特点。对教学具有一定的指导意义。然而，这些研究也有局限性。生成的概念图反映了所有学生的学习成绩。所有学生都遵循相同的学习路径和学习时长。进一步分析概念图通常需要劳动力的帮助。它很费时，不利于从概念图中快速提取信息。

总体上，对概念图的研究归纳为两种类型。第一类研究利用文本分析[26]技术自动生成概念图。然而，生成的概念图缺乏进一步的分析，不能反映学生的学习成绩，也不能反映自适应学习系统的特点。第二种研究考虑了学生的学习成绩。然而，生成的概念图反映了所有学生的特点，不能根据学生对概念的掌握程度来区分学生。该诊断也适用于所有学生，并没有能力区分学生。在劳动力的帮助下，概念图的分析是费时的。本研究采用聚类算法，根据学生对概念的掌握程度将学生进行分组，并结合关联规则挖掘方法生成多个概念图。每个概念图对应一组学生的学习特征，可以有效区分学生。利用拓扑排序算法，分别分析概念图，自动生成具有不同学生学习特征的学习路径。每个学习路径的学习顺序和学习时长不同，可以为教师提供教学建议，为学生提供学习指导

LPG生成算法

LPG (Learning Paths Generation)算法利用学生的考试记录，结合问题与概念之间的关系数据，将学生对问题的掌握转化为学生对概念的掌握，提取学生的学习特征。聚类方法是根据学生的学习特点将学生分成不同的群体，然后通过关联规则挖掘方法生成多个不同的概念图。在这些概念图的基础上，利用拓扑排序算法分析概念之间的关联，不同的概念图可以生成不同的学习路径。这些学习路径是根据每一组学生对每个概念的掌握程度来简化的。可以根据学生的学习情况生成不同的简化学习路径。学生可以根据这些简化的学习路径，有针对性地学习概念。LPG算法流程图如图1所示。此外，我们在本文中使用表1中的符号

结论

现有的概念图研究对学生的学习成绩研究不足，研究结果是宏观的，不能反映自适应学习系统的特点。针对这些局限性，提出了一种基于概念图的自适应学习系统学习路径自动生成算法LPG。LPG算法利用聚类技术，根据学生对概念的掌握程度，自动将学生分成若干组，并结合关联规则挖掘方法生成多个概念图，利用拓扑排序算法生成学习路径。

具有学习时间的简化学习路径可以为教育者提供学生的学习诊断报告和相应的学习内容推荐计划。对学生进行适应性评估，并给出相应的学习路径。随着小考数量的增加，可以不断地对学生进行诊断，并根据学生的考试记录自动更新学习路径。教育者可以根据最新的学习路径不断调整教学计划，学生可以进一步提高学习效率。本研究可为适应性学习系统的教学提供建议和规划。随着数据的不断收集，可以及时更新自适应学习系统，进一步提高适应性。

实验表明，LPG算法具有以下特点:1)适应性好，学生群体间的差异性高。基于学生对概念的掌握，可以产生几种不同的学习路径。2)劳动力成本低。学习路径可以自动生成和简化。3)高灵活性。学习路径的数量可以根据实际需要设置。

LPG算法虽然在自动生成学习路径方面表现良好，但也存在一定的局限性。受时间、空间等因素的限制，本文只验证了LPG算法能够区分学生，没有观察学生的长期学习表现，也没有验证LPG算法对提高学生成绩有帮助。此外，为了方便提取学生的学习特征，问题与概念之间的关系只能是一对一的，这降低了试卷的灵活性。从大时间跨度中收集数据，使LPG算法适用于多对多关系数据是我们未来的工作
