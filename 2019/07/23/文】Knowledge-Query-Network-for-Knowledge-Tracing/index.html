Knowledge Qery Network for Knowledge Tracing
– How Knowledge Interacts with Skills
Jinseok Lee & Dit-Yan Yeung
Hong Kong University of Science and Technology

KQN使用神经网络来编码学生的学习活动,分为知识状态和技能向量，并通过这两种向量之间的点积对交互进行建模。通过为此，我们引入了一个新的概念，称为概率技能相似性，它将技能向量之间的成对余弦和欧氏距离与相应技能的优势比联系起来，使KQN具有可解释性和直观性。

在四个公共数据集上，

  • 基于预测精度，KQN优于现有的KT模型。
  • 知识之间的互动状态和技能可以可视化进行解释。
  • 基于概率技能相似性，一个技能域可以利用KQN的技能向量之间的距离聚类分析。
  • 对于不同的向量空间维数，KQN预测精度高，技能向量的距离矩阵之间具有较强的正相关关系。

介绍

智能辅导系统和大型开放式在线课程的优势之一是，它们可以在一个自适应的学习环境中监控和跟踪学生活动，从而发挥学习者建模的作用。
学习者模型提供对学生状态的估计，包括两个相互关联的方面:领域建模和知识建模。
领域模型研究问题域中的结构(它确定了问题与哪些技能相关)。领域模型的另一个任务是发现技能领域的结构，可以手动或自动执行。
另一方面，抽象意义上的知识模型,跟踪学生在解题过程中的知识。知识以知识状态的名义以各种形式被描述，而知识状态还没有普遍的定义。在本文中，知识状态这一术语被用来描述一个学生的技能获得的总体水平。
通常，领域建模和知识建模是分开的。然而，我们试图提供两种方法，学生的问题解决记录可以成为确定技能领域潜在结构的重要输入特征。

KT是一个分析学生活动和学习知识获取的研究领域，其主要任务是描述学生的知识。要详细说明，请考虑一个解决了一系列问题的学生。然后，学生的数据由元组的时间序列给出，每个元组由与每个时间步的问题相关的技能和表示学生是否给出正确答案的二进制正确性组成。通过调用这样的元组学生响应，KT问题可以表述为:

  • 给定学生在时间步t之前的回答，描述学生在当前时间步t时的知识状态
  • 给定下一个时间步t + 1的技能，通过对学生在t时刻的知识状态与t + 1时刻的技能之间的交互进行建模，预测其正确性，我们称之为知识交互。

因此，KT模型的质量是通过描述学生知识状态的能力和预测正确性的准确性来衡量的。

此外，由于知识交互建模是为了描述学生的知识状态对不同技能的反应，因此KT模型能够解释从知识交互中推断出来的技能之间的关系。例如，如果一个模型观察到学生在学习整数的时候没有学习整数的减法，那么我们可以说“整数的加法”独立于“整数的减法”。同样，如果学生对一项技能的知识状态的改变影响到另一项技能的知识状态，那么这两项技能也会相互依赖。我们认为，对这种技能关系进行建模可以进一步探索技能域的潜在结构，这也是领域建模的主题。

然而，现有的KT模型提供了有限的知识状态或知识交互，或两者都有。例如，Bayesian Knowledge Tracing (BKT)对知识状态强加了一个二元假设，限制太多，难以直观，而Deep Knowledge Tracing (DKT)没有对知识交互做出解释。本文提出了一种新的神经网络KT模型，即知识查询网络(KQN)，该模型能较好地概括知识状态和解释知识交互。其核心思想是利用知识状态向量和技能向量之间的点积来定义知识交互，同时利用神经网络将学生的反应和技能编码为具有相同维数的向量。此外，我们引入了一个新的概念，称为概率技能相似性，它将技能向量之间的余弦和欧氏距离与相应技能的优势比联系起来。在此基础上，利用聚类分析方法，探讨了技能域的潜在结构。最后，通过比较不同向量空间维数下的预测精度和技能向量的距离矩阵，证明了KQN在预测正确性和学习技能向量方面是稳定的。

相关工作

项目反应理论(IRT)是一个用来建模问题与正确性之间关系的框架。其最简单的形式是使用逻辑回归模型来估计学生的效率和技能差异。但是，它假设profciency是常量，并且没有解释任何问题的结构。为了克服这些限制，贝叶斯扩展的IRT提出了一个分层结构的项目(HIRT)和学生的知识状态(TIRT)的时间变化。尽管如此，HIRT假定学生的专业能力不变，而TIRT缺乏领域分析的能力。

在BKT中，学生的知识状态被看作是一组二元潜在变量，每个技能一个，有两个可能的状态，已知的和未知的。然后，一组可观察变量，每一个对应于每项技能的正确性，都以这组二进制变量为条件。例如，假设我们有一个贯穿本文的学生Ben的运行示例，他的记录如表1所示。因此，技能1和技能2有两个知识状态和正确性变量，共四个变量，两个独立的BKT模型，输入数据如表1b所示。然后对BKT中的知识获取进行马尔可夫模型建模，其中知识交互由一组可解释方程控制。由于BKT缺乏遗忘和个性化的能力，已经提出了一些扩展。然而，最重要的是，BKT对不同技能的独立性假设被认为是高度受限和无效的，其中模型不能利用整个数据。例如，Ben对技能1的回答历史不能告诉他对技能2的回答，因为每个技能应该有两个单独的模型。

学习因素分析(LFA)[2]将学生的知识状态建模为一组二元变量，每个二元变量对应一个技能。使用逻辑回归模型预测每个时间步骤的正确性，该模型具有与学生、技能相关的协变量，以及学生反应的汇总统计，即，每项技能过去的机会数量。

性能因素分析(PFA)[16]扩展LFA通过将“每个技能的机会数”到“正确答案的数量/技能”和“不正确的答案的数量每技能”与其他相同的,例如,在本的情况下,输入数据预处理如表1所示表1c和d。

由于在LFA和PFA中，对正确性的估计是用学生协变量来解释的，而在没有变量交互作用的情况下，使用技能-特殊fc协变量来解释，所以这两个模型并没有描述学生关于一项技能的知识状态如何受到另一项技能的知识状态的影响;相反，学生参数，也称为学生效率，是联系不同技能的知识状态的唯一因素。此外，技能是由技能-特殊变量的回归系数来解释的，我们不能直接从这些系数来判断技能域的结构。

DKT作为第一个神经网络KT模型，以学生的反应作为输入，将学生的知识状态编码为一个由一个递归神经网络(RNN)计算得到的摘要向量，RNN是一种著名的时间数据建模神经网络技术。然而，DKT并不直接定义知识交互。详细地说，每个时间步的学生响应都形成一个元组(qt, at)，其中qt和at分别表示问题ID和正确性。例如，表1a中Ben的原始数据在t = 1时表示为(1,0)。然后将输入传递给RNN层，在原始论文中使用了长短时记忆，其输出隐藏状态经过afne变换后传递给logistic函数，即,=σ(W·ht + b),在ht是输出LSTM层和σ的隐藏状态以聪明元素物流功能。最后,欧美的k元素是用来预测正确性在下次一步考虑到下一题ID是k。尽管其优越的预测性能在现有的经典方法,DKT遭到了其他报纸的批评，在教育应用程序缺乏实用性。这是因为输出的隐藏状态ht在本质上很难被解释为知识状态，并且该模型并没有提供对知识交互的洞察。

为了使神经网络KT模型更具解释性，通过扩展记忆增强神经网络(MANNs)引入动态键值记忆网络(DKVMN)。与DKT一样，DKVMN使用原始数据作为输入。DKVMN将学生反应的时间信息累积到一个动态矩阵中，即值记忆，同时将技能嵌入到一个静态矩阵中，即键记忆。DKVMN利用余弦相似度计算的注意权值来定义值向量与关键向量之间的交互关系，并通过将一个加权的值向量和一个嵌入的技能向量串联到一个多层感知器(MLP)来预测正确性。虽然DKVMN的预测精度已经被证明高于DKT，但是模型的输出使用MLP仍然很难解释知识交互。

模型

动机

为了在直观描述知识交互的同时概括知识状态，我们提出了一个模型，该模型将学生的知识和技能投射到相同的嵌入维数d的向量空间中。它们有单位长度和正坐标。利用当前知识状态向量与下一题技能向量的点积，给出了答对的概率估计的logit。这需要知识状态和技能向量都位于同一个向量空间。

现在，我们来说明为什么技能向量被设置为单位长度，并被约束为一个正值——前者使logit只依赖于相关技能向量的方向，而后者确保了对一项技能的学习不会减少对另一项技能的学习。

例如,在一个二维向量空间,两次答题之后，t=2时，假设知识状态KS2 = (1, 1) T。三个技能向量,s1 = (1,0) T和s2 = (0,1) T 对应技能1和2,s3 = (−1,0) T对应第三个“xxx”技能,如图1所示。当t = 3时，他正确地回答了技能1，他的知识状态可能变为KS3 = (2,1) T。然后对s1的logit值从KS2·s1 = 1增加到KS3·s1 = 2，而对技能2的logit值与KS2·s2 = KS3·s2 = 1相同。但是，技能3的logit值会从KS2·s3 = - 1下降到KS3·s3 = - 2，从而降低了对技能3正确性的概率估计。这是违反直觉的，因为我们正在处理的数据集具有同一领域内的一组技能，例如数学。

结论和未来工作

本文的贡献如下

  • KQN在正确率预测的准确度方面比其他模型优秀
  • KQN能够根据不同的技能对学生的知识状态进行查询，有助于通过可视化的方式解释知识交互。
  • KQN的结构引入了概率技能相似性的概念，将两个技能向量之间的余弦和欧氏距离与本文前面介绍的相应技能的优势比联系起来。这使得技能向量及其两两距离对于领域建模(例如，使用聚类分析)非常有用。
  • KQN强劲到向量空间的维数变化的知识状态和技能向量,其预测精度不退化,而且学习强烈正相关集技能之间的两两距离向量维数的值是不同的,或者说,KQN学习技能稳定之间的潜在关系。

与其他神经网络模型相比，KQN有更多的参数需要学习。例如，由于在技能编码器中包含一个MLP，而在知识状态编码器中包含一个RNN，因此KQN在计算上比DKT更复杂，而DKT只有一个RNN用于编码学生的响应。启发式地，与DKT+KQN相比，训练KQN需要更多的GPU内存。尽管如此，我们认为上述KQN的优点足以弥补空间复杂度的增加。

KQN提出了一种解决KT问题的新方法，即在同一向量空间中对知识状态和技能向量进行重构。它将知识状态和技能作为向量的一般形式，同时将知识交互清晰地刻画为两类向量之间的点积。从两两之间的距离的技能向量是解释为相应的优势比的对数从概率的角度来看,这些距离可以成为有用的特性域建模探索技能领域的潜在结构,可作为KT的未来研究的方向。
